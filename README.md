# Student Performance Prediction

This is a simple Machine Learning project that predicts a student's performance based on:
- Hours studied
- Attendance
- Previous exam score
- Extra classes

## How to Run
1. Install requirements:
pip install -r requirements.txt
2. Run the project:
python student_performance.py

## Files
- `student_performance.py` → main ML script  
- `data/students.csv` → dataset  
- `model/student_model.pkl` → saved model  
- `requirements.txt` → dependencies  

## Sample Output
- Model trains successfully  
- Shows MAE and R2 Score  
- Predicts performance for a new student  

## Tech Used
- Python  
- Pandas  
- Scikit-Learn
